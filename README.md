# drive-dl

### Usage
```shell
# Clone and enter into repo.
~$ git clone https:github.com/sil-car/drive-dl.git
~$ cd drive-dl
~/drive-dl$

# Create virtual environment.
~/drive-dl$ python3 -m venv env

# Activate virtual environment.
~/drive-dl$ source env/bin/activate

# Install enviroment dependencies from requirements.txt.
(env) ~/drive-dl$ pip3 install --requirement requirements.txt

# Run the app.
(env) ~/drive-dl$ python3 download-drive-file.py

# Deactivate environment afterwards.
(env) ~/drive-dl$ deactivate
~/drive-dl$
```
